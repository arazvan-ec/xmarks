## What changed

This is the first release of the git-native memory design
(`docs/research/git-native-memory-design.md`, proposal P2): typed entries,
budgeted relevance-based injection, and on-demand recall — still pure
markdown, no database or worker.

- **Typed ledger entries.** `/flywheel:compound` now writes one entry per
  distinct learning as `## <type>: <title>` (`type` is `decision` / `gotcha` /
  `pattern` / `bugfix`) followed by a single greppable metadata comment
  (`<!-- fw: type=…; date=…; files=…; spec=…; pr=…; branch=… -->`) and a short
  prose body, instead of one bundled dated entry with free-prose bullets.
- **Relevance-scored, budgeted SessionStart injection.** `scripts/session-start.sh`
  no longer dumps the first ~50 lines of the ledger. It now computes the
  current branch, changed files (vs. the default branch plus uncommitted), and
  active spec via `git`, scores each entry (+3 files overlap, +2 branch/spec
  match, +1 recency within 30 days), and injects only the top **N=12** full
  entries (override with `FLYWHEEL_LEARNINGS_INJECT`), followed by a one-line
  pointer for the rest. Still read-only, fail-open, and exits 0 on a missing or
  unparsable ledger.
- **New `/flywheel:recall <query>`.** Greps entry headers and metadata for a
  cheap numbered list of matches, then expands one entry's full body on
  request — the progressive-disclosure layer that makes the smaller
  SessionStart injection safe. Added to the README command table and
  `/flywheel:help`.
- **Backward compatible.** Old free-prose entries (`## <date> — <feature>`,
  no metadata comment) still load in both SessionStart and `/recall`, treated
  as always-eligible, low-priority entries. Nothing is rewritten or deleted.
- **Deferred to later releases** (per the locked design): a `.tsv` index,
  semi-auto staging via a Stop hook, ledger rotation/archival, and interop
  with claude-mem/Engram. None of those are needed for this release to work.

No new binary/dependency — `git`, `grep`, and `awk` only, matching what the
hook already assumed was available.
